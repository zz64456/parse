# Parse to Download

使用 selenium, requests 下載 指定html裡的指定圖片

## 安裝 Python3

請先確定電腦有安裝 [Python](https://www.python.org/)

### 安裝 Python3 套件

```
pip3 install -r requirements.txt
```

### 環境設定
